Real life RPG 1

I wanted to do something special, this was a little game where you could buy houses, hang out and do all kinds of other activities, but eventually, the world would end, and you would be teleported to a shop where you could buy in-game cosmetic items and more with stuff you earned while playing the game. 

I added more content to the game as time went on, but i decided to do it in a unique way, for the most important updates i decided to do live events, similar to those you would see in games like Fortnite, these would be one-time events that would build up lore and story and also update the map with more content. I recorded the last live event on YouTube.

A problem with these live events would be that they weren't officially supported by the engine and game platform i used (LittleBigPlanet) so i had to do some workarounds to get it to work (i even faked interaction between other players on the server), there wasn't actually a server being updated live, in reality i updated the whole game a bit before the announced live event would start, knowing that the players of the old version will have been kicked out of the game by now, since the game ends eventually, as i mentioned. So that new build would contain the live event and after the live event is over the players would either be transfered to an unlisted version of the newest build with all of the changes made by the live event being in effect, the new map etc, or the changes would just be done on the live event build they're currently on too, then i updated the build while they were still in the game to the post-live event version, so if they'd leave the game and rejoin they woudn't see the live event again and instead the aftermath of the event that happened.

The problem with this is: what would happen if a player joined right before the live event was to start but after i already updated to the live event build, they would be slightly out of sync, but thats not even that bad, the real issue lies within what would happen if someone decided to restart the level or leave it during the event and rejoin, the live event would repeat!

Anyway, i dont think that ever really happened though, so thats good. Eventually the final live event lead up to the entire map being destroyed and Real life RPG 2 to come out!
