Choosing an Engine

Welcome to my first blog post!

So i have decided that im going to be primarily using Gamemaker Studio 2
for 2D games, and Unreal Engine for 3D games. I have tested Gamemaker
for a while now and i find it to be easy and convenient to use,
it also only requires a one-time fee to start commercially selling your games.
Most engines are going to take a cut once your game has sold a certrain number
of copies, but not Gamemaker (or Godot). But the downside to Gamemaker is that
you can't create 3D games, at least not without tricking the hell out of
the engine. So im going to be using Unreal for 3D, i've been considering Unity,
however since the whole drama that happened where they wanted to charge game
devs for installations, i don't think that's a good idea.

Both Unreal and Unity feature a visual programming language, similar to the one in used back in LittleBigPlanet. So i'd probably be able to get into it pretty quickly, however
for Gamemaker at least i do want to use its dedicated gml code as much as possible,
because it would improve my traditonal coding skills, for Unreal i think i'll use
their blueprint system primarily though, as they only offer c++ as an alternative
which im not very interested in learning right now, blueprints seem to be powerful enough.

I had also been learning Renpy a while back, it's an engine and programming language for the developement of visual novels and it's very easy to use, i feel like i got the most
importing things figured out about it, right now i dont really want to focus on visual
novel stuff though, but it's nice to know i suppose.

Alright so i'm currently watching a bunch of tutorials on Gamemaker, i already learned some stuff but its all still very basic. And without tutorials i cant really make much at all,
so im kind of trapped in tutorial hell right now. I'll try to participate in Game Jams
as soon as possible as its the quickest and best way to learn in game developement from what i've seen.

But first i want to finish a short experimental Platformer game first called Rainhiker,
so i can gain some more skill before actually participating in any Game Jams.
I only have a rough concept for the game right now i do think i can make something
out of it...
