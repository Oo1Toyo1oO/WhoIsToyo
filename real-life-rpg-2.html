Real life RPG 2

After the live event that happened back in Real life RPG 1 this sequel came out, featuring a brand new map and a bunch of new features, an old friend named InfinityZockerA9 helped me a bit with it, but i did all of the programming and most other things, this is one of the biggest game's i've ever made, it has so much content we had serious stability issues, and even now the game always crashes when you're playing with more than 2 people, for a while we coudn't even get into the game ourselves because it was too overloaded with stuff.

This game features a better looking map than the first one, house upgrades, dreams, a mine, upgrades to buy, secret minigames and a lot more, it even saves your progress now and the game doesn't end anymore after a certain time limit, meaning if you buy a house for example, it will still be there the next time to play the game.

We even used to have npc's that move around the map but we had to scrap it as it would make the performance even worse. There were never any life events for this part though, the game is pretty much a finished, unevolving work.
